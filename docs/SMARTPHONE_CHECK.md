# スマホ表示の確認（6.2）

スマホ用の表示は**すでに実装済み**です。このチェックは「実機やブラウザで問題ないか確認する」ためのものです。

---

## すでに入っているスマホ対応

- **ボトムナビ** … スマホ幅では画面上部の「記録する」「マイ記録」などの代わりに、**画面下に固定のボトムナビ**（記録・マイ・ランキング・設定）が表示されます。
- **レスポンシブレイアウト** … カード一覧はスマホでは 1 列、タブレット以上で 2 列になります。
- **余白** … メインコンテンツの下に余白を多めにとって、ボトムナビに隠れないようにしています。

---

## 確認のしかた

1. **ブラウザの開発者ツール**  
   Chrome などで F12 → デバイスツールバー（スマホアイコン）をオン → 機種を「iPhone 12」などにすると、スマホ幅で表示されます。

2. **実機**  
   同じ Wi‑Fi のスマホで `http://（PCのIP）:3000` を開く（例: `http://192.168.1.10:3000`）。  
   デプロイ後は本番 URL をスマホで開いて確認。

---

## 確認してほしいポイント

| 項目 | 確認 |
|------|------|
| ボトムナビが画面下に表示され、タップで画面が切り替わる | ☐ |
| トップ・記録・マイ・ランキングの内容が読みやすい（はみ出し・重なりなし） | ☐ |
| 全件一覧の絞り込みプルダウンがタップしやすい | ☐ |
| マップが表示され、ピンや「住所から位置を取得」が操作できる | ☐ |
| 縦向きでスクロールして全体が使える | ☐ |

問題があればメモして修正すると、そのまま 5（デプロイ）に進めます。
